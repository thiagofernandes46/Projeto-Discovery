GUIA RÁPIDO – COMANDOS CSS MAIS IMPORTANTES (INICIANTE)

RESET BÁSICO
Remove margens e padroniza o comportamento dos elementos.

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
  }

---

CORES E FUNDOS

color: white;              → cor do texto
background: black;         → fundo
background-image: url();   → imagem de fundo
background-size: cover;    → cobre toda a área
background-position: center;

---

TAMANHO E ESPAÇAMENTO

width: 300px;
height: 200px;

padding: 16px;   → espaço interno
margin: 16px;    → espaço externo

---

BORDAS

border: 1px solid black;
border-radius: 8px;   → arredondamento

---

FONTES E TEXTO

font-family: Arial, sans-serif;
font-size: 16px;
font-weight: bold;
text-align: center;
text-decoration: none;

---

FLEXBOX (ALINHAMENTO)

display: flex;
align-items: center;      → alinhamento vertical
justify-content: center;  → alinhamento horizontal
flex-direction: column;   → coluna
gap: 16px;                → espaço entre elementos

---

HOVER (INTERAÇÃO)

button:hover {
background: gray;
}

Serve para efeitos ao passar o mouse.

---

TRANSIÇÕES

transition: 0.3s ease;

Deixa mudanças suaves (cores, posição, tamanho).

---

POSICIONAMENTO

position: relative;
position: absolute;

top: 0;
left: 0;

Usado para posicionar elementos com precisão.

---

RESPONSIVIDADE

@media (min-width: 700px) {
body {
background: red;
}
}

Permite adaptar o layout para telas maiores.

---

VARIÁVEIS CSS

:root {
--cor-principal: red;
}

color: var(--cor-principal);

Facilita manter padrão de cores e temas.

---

SOMBRAS E EFEITOS

box-shadow: 0 10px 30px rgba(0,0,0,0.2);

backdrop-filter: blur(4px);

Usados para dar profundidade ao layout.

---

DICAS IMPORTANTES

1. Primeiro faça para celular (mobile first).
2. Use flexbox para alinhar.
3. Evite repetir código (use variáveis).
4. Organização do CSS é tão importante quanto o design.
5. Pequenas animações deixam o site muito mais profissional.
